# Bank Account System Using OOP(Python)


## Description
This project demonstrates a simple Bank Account system implemented using Object-Oriented Programming concepts in Python.

## OOP Concepts Used
  - Class and Object
  - Constructor ('__init__')
  - Encapsulation (private PIN)
  - Methods for banking operations

## Features 
   - PIN authentication
   - Check Balance
   - Deposit Money
   - Withdraw Money

## Technologies used 
   - python 3

## How to Run
    ```bash
        python bank_account.py
